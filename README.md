## 一个简单的webpack脚手架项目

## dev
> $ npm start

## build
> $ npm run build


## 生成 jest config
```sh
$ ./node_modules/.bin/jest --init
```

## 生成 tsconfig
```sh
$ ./node_modules/.bin/tsc --init
```
