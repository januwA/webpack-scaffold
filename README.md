## Simple webpack scaffold

## dev
```sh
$ npm start
```

## build
```sh
$ npm run build
```


## init jest config
```sh
$ ./node_modules/.bin/jest --init
```

## init tsconfig
```sh
$ ./node_modules/.bin/tsc --init
```
